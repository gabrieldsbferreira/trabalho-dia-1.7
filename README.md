# trabalho-dia-1.7
Trabalho de aplicação Kotlin com banco local
Projeto: App Lista de Tarefas em Kotlin com Room, entrega dia 01/06.
Grupo: Gabriel, Kauã Marley, Dhyan e Douglas (3-A)

Requisitos cumpridos: 
Kotlin, banco local Room/SQLite, 3 tabelas relacionadas: usuarios, categorias, tarefas. 

Relacionamentos: 
Tarefa ligada a 1 Usuario e 1 Categoria. Se deletar Usuario apaga as tarefas. Se deletar Categoria só tira o vínculo.

Como rodar: Clonar do GitHub, abrir no Android Studio, sincronizar Gradle, executar em Android 7.0+.

Tecnologias: Kotlin, Room 2.6.1, ViewModel, LiveData, Coroutines.
